# 🌍 EcoSphere AI

Smarter habits for a sustainable future.

## About the Project
EcoSphere AI is an AI-powered sustainability platform developed as part of the Lenovo LEAP NextGen Scholar Capstone Project.

The project supports:

- SDG 2 – Zero Hunger
- SDG 13 – Climate Action

## Features

### Website
Provides awareness about:
- Food management
- Energy saving
- Water conservation
- Sustainable habits
- Eco Score

Website Link:
https://ecosphere-ai-sustain.lovable.app/

### AI Assistant
Built using Botpress and provides:
- Food waste reduction tips
- Leftover recipe ideas
- Recycling guidance
- Sustainable living suggestions
